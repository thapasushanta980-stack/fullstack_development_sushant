What is Full‑Stack Development?

Full‑stack development is the practice of building both the front end (what users see and interact with in the browser) and the back end (the server, database, and application logic). A full‑stack developer understands how data flows from a database and API to the UI, and how the UI sends requests back to the server.

  • Front end: HTML (structure), CSS (presentation), JavaScript (interactivity) and UI libraries like Bootstrap.
  • Back end: Server frameworks (e.g., Node/Express, Django), databases (SQL/NoSQL), authentication, and APIs.
  • DevOps: Version control (Git), deployment, performance and accessibility best practices.
Goal for this portfolio: demonstrate solid understanding with thorough, consistent evidence that maps to an “Excellent” rubric rating.
